
# 📘 Projeto 2 - Portfólio de Imagens

## 🎯 Propósito
Elaborar uma galeria de imagens organizada como portfólio, com **fotos e descrições** usando tags semânticas.

## 🛠️ Ferramentas Utilizadas
- **Editor**: VS Code  
- **Linguagem**: HTML5  

## 📂 Estrutura do Projeto
- `index.html`

## 📝 Conteúdo da Página
- Cabeçalho principal com título.  
- Seção dedicada a uma lista de **filmes de comédia**.  
- Imagens representadas em `<figure>` acompanhadas de legendas (`<figcaption>`).
